# mini_project
Smart Attendance system Project 

This project is built using Arduino, GSM and Bluetooth module for gps location

Students Gps location is shared to arduino using bluetooth if the student is present then he is marked as present or absent based on his location

start trigger is given using GSM module a message nedd to be sent to the registered mobile number using the keyword set 

This project can be improved using database or cloud storage to store the attendance of students and can get report of that using a keyword
